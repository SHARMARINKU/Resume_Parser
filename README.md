# Resume Parser

**Resume_Parser** is a Python project that uses **LangChain** and LLM (Large Language Models) to parse resumes (CVs) and extract structured information. It supports multiple formats and can intelligently identify key fields such as name, education, skills, work experience, and more.

## Features

- Parse resumes in **PDF**, **DOCX**, or **TXT** formats
- Extract key details: **name**, **email**, **phone**, **education**, **skills**, **experience**, etc.
- Use of LLMs + LangChain for more accurate, intelligent parsing
- Configurable parsing logic via rule-based + model-based approaches
- Export parsed data to JSON or other structured formats
